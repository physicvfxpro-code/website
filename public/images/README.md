# Images

```
images/
├── hero-bg.jpg          # Hero section background (recommended: 1920x1080)
├── cta-bg.jpg           # CTA section background
├── og-image.jpg         # Open Graph image for social sharing (1200x630)
├── portfolio/
│   ├── project-1.jpg
│   ├── project-2.jpg
│   ├── project-3.jpg
│   ├── project-4.jpg
│   ├── project-5.jpg
│   └── project-6.jpg
├── shop/

│   ├── pack-icons.jpg
│   ├── pack-starter.jpg
│   ├── pack-presets.jpg
│   ├── pack-video.jpg
│   └── pack-luts.jpg
└── tools/
    ├── camera-1.jpg
    ├── camera-2.jpg
    ├── drone-1.jpg
    ├── gimbal-1.jpg
    ├── gimbal-2.jpg
    ├── light-1.jpg
    ├── light-2.jpg
    ├── light-3.jpg

    ├── audio-2.jpg
    ├── sw-1.jpg
```
